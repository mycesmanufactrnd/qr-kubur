## `_redirects` File Overview

The `public/_redirects` file is used by specific static web hosting services such as **Netlify** and **Cloudflare Pages** to define URL redirection rules for your website.

### Why it matters

- Controls how visitors are redirected between URLs  
- Helps search engines find the correct pages (SEO)  
- Essential for **Single Page Applications (SPAs)** to handle client-side routing properly  

### Common use cases

- Redirecting old URLs to new ones  
- Handling 404s and fallback routes  
- Enforcing HTTPS or canonical domains
